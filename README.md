## Deskripsi

Aplikasi WEB sederhana ini dibuat untuk UTS Pemrograman Web 2. Aplikasi WEB sederhana ini digunakan untuk mengelola data kategori buku dengan fitur tambah, lihat, edit, dan hapus data (CRUD).

## Teknologi

- PHP Native
- MySQL
- Bootstrap 5
- XAMPP

## Cara Menjalankan

1. Install XAMPP, pastikan Apache dan MySQL sudah berjalan
2. Copy folder project ke C:\xampp\htdocs\uts_perpustakaan_[60324085]
3. Buka phpMyAdmin di http://localhost/phpmyadmin
4. Buat database baru bernama uts_perpustakaan_[60324085]
5. Import file database/database_backup.sql ke database tersebut
6. Buka browser dan akses http://localhost/uts_perpustakaan_[60324085]
